# 背景
- M5Stack-CoreS3-Lite で開発を進めたい
- 現状何ができるか分かっていない
- とりあえずメジャーな遊び方をしたいが, 何ができるのかも分かっていないので調査, 整理しながら提案してほしい

# 開発方針
- M5Stack-CoreS3-Lite で実装できることを調査し, 新しいものは research ディレクトリに md 形式でまとめる
- TDD + アジャイル開発的に取り組み, まずは小さく動くものから実装していく
- 認知負債を軽減するため, 実装が完了した場合, summary ディレクトリに概要を md 形式でまとめる
- Issue の起票 → 実装 → PR → レビュー → マージ の一連の流れを Claude が行う
- ハードウェアに関して分からないことを解説した場合, knowledge ディレクトリに md 形式で分からなかったことと解説についてまとめる

# 技術選定
- バージョン管理は git で行う
- gh コマンドを使用する
- Issue やまとめの生成時に図示をした方が整理できる場合, mermaid を用いる
